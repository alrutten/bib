Where are the data?

Data is contained in a database server at scidb.orn.mpg.de, in order to access it you need to remember:

HOST: scidb.orn.mpg.de USERNAME: bt PASSWORD: bt

How can I access the data?

There are several interfaces which can be used to enter the data. Although they are rather different, all interfaces connect to the same database.
Excel!
  • Go to MySQL Installer click download, then "No thanks, just start my download".
  • Double-click on "mysql-installer-community" go to "Add Product and Features ", choose "Applications", then choose "MySQL For Excel.".
  • Open "Excel", go to "Data" and click "MySQL for Excel".
  • Go to "New connection", enter "Host", "User" and "Password" and give a name to your connection (e.g. "BT data entry").
  • Then go to "remote connections", click on "FIELD_BTatWESTERHOLZ" and then select a table (most likely NESTS).
  • In order to edit and add new data click on "Edit MySQL data". To save the data to database push "Commit Changes".
Heidy SQL!
  • Download from here and install.
  • Open Heidy SQL, create a new connection in the Session manager, give it a name and save it.
  • Open the database and choose a table to edit.
Open office!
  • Assusming you have Open Office installed you will need to install "MySQL Connector" from here.
  • After you install the connector (double click on it and follow the instructions), go to "Database", choose "Connect to an existing database" scroll all the way down and choose "MySQL".
  • Choose "connect directly" and enter the Host name, username and password.
  • Open the database and choose a table to edit.
